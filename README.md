# Numeration_files
## Как запускать?
Скопируйте команды и вставьте в терминал
### Windows
```python
Python -m venv .venv
.venv/scripts/activate
pip install -r requirements.txt
Python main.py
```

### Linux
```python
Python -m venv .venv
.venv/source/activate
pip install -r requirements.txt
Python main.py
```
